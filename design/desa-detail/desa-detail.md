# Hero Section Design Reference — Desa Wisata Landing Page

## 1. Overall Design Direction

Create a premium, modern, and elegant hero section for an Indonesian tourism village website. The design should feel warm, natural, editorial, and culturally authentic. It should combine modern landing page composition with a soft village tourism atmosphere.

The visual style must feel professional and production-ready, not generic, cluttered, or AI-generated. Use a calm natural color palette, refined typography, realistic photography, and spacious layout.

## 2. Layout Structure

Use a full-width desktop hero layout with a balanced two-column composition.

The left side should contain the main brand message, headline, supporting text, and CTA buttons. The right side should contain a large visual collage of village tourism images.

Recommended layout:

* Hero section height: around 850–950px
* Max content width: 1440–1560px
* Left content width: around 40–45%
* Right visual collage width: around 55–60%
* Use generous horizontal padding, around 64–96px
* Align content vertically around the center of the hero section
* Keep enough whitespace between text and image collage

The layout should feel asymmetric but balanced. The text block should be visually strong, while the image collage should create emotional appeal.

## 3. Header / Navigation Rules

Place a clean navigation bar at the top of the hero section.

Header layout:

* Logo on the far left
* Main navigation links centered
* Language selector and CTA button on the right
* Use enough spacing between navigation items
* Keep the header minimal and elegant

Navigation style:

* Active menu item should use a muted green color
* Add a small underline or indicator below the active menu
* Use clean sans-serif typography
* CTA button should use a rounded pill shape
* Avoid heavy borders or excessive shadows

Example navigation items:

* Home
* About Village
* Destinations
* Activities
* Local Products
* Gallery
* Blog

## 4. Typography Rules

Use a strong editorial typography combination.

Headline:

* Use a high-contrast serif font
* Large size, around 72–96px on desktop
* Tight line height, around 0.95–1.1
* Use strong hierarchy and elegant spacing
* Highlight important words such as “Desa Wisata” using muted green and italic serif styling

Body text:

* Use a clean sans-serif font
* Size around 18–20px
* Line height around 1.6
* Use dark gray instead of pure black
* Keep the paragraph width around 520–620px

Navigation and buttons:

* Use a modern sans-serif font
* Medium weight
* Keep text readable and clean

## 5. Color Palette Rules

Use warm natural tones inspired by Indonesian village tourism.

Recommended colors:

* Background: warm ivory, soft cream, or off-white
* Primary text: deep charcoal or warm black
* Brand green: muted forest green or olive green
* Accent color: soft terracotta, warm brown, or natural beige
* Secondary background: light sand or pale beige
* Border color: soft beige or light warm gray

Avoid:

* Neon colors
* Overly saturated gradients
* Harsh black and white contrast
* Random colorful decorative elements
* Excessive glassmorphism

## 6. Image Collage Rules

The right side should use a layered image collage with rounded corners.

Image composition:

* One large main image as the visual anchor
* Two medium supporting images on the right
* One smaller supporting image near the bottom-left of the collage
* One circular badge or decorative seal overlapping the images
* Images should overlap naturally but remain readable

Image style:

* Use realistic high-quality photography
* Show Indonesian village scenery, rice fields, mountains, traditional houses, local culture, local food, crafts, or village activities
* Use warm natural lighting
* Keep all images visually consistent in tone and color grading

Card styling:

* Use large rounded corners, around 28–40px
* Add a thin light border or white stroke
* Use soft shadows for depth
* Avoid harsh drop shadows
* Maintain clean spacing between overlapping image cards

## 7. CTA Button Rules

Use two CTA buttons below the description text.

Primary button:

* Filled muted green background
* White text
* Rounded pill shape
* Include a simple arrow icon on the right
* Make it visually dominant

Secondary button:

* Transparent or light background
* Thin beige border
* Warm brown or muted text color
* Rounded pill shape
* Include a simple arrow icon

Button sizing:

* Height around 56–64px
* Horizontal padding around 28–40px
* Gap between buttons around 20–28px

## 8. Decorative Elements

Use subtle decorative details to support the village tourism feeling.

Allowed decorative elements:

* Small leaf icon near the eyebrow text
* Thin dashed curved line near the CTA area
* Small botanical illustration
* Soft contour map lines in the background
* Very subtle watercolor-like texture

Rules:

* Decorations must be subtle and not distract from the main content
* Use low opacity
* Keep the design clean and premium
* Avoid random floating shapes or excessive ornaments

## 9. Visual Hierarchy

The hierarchy should be clear:

1. Logo and navigation
2. Main headline
3. Supporting description
4. CTA buttons
5. Image collage
6. Decorative elements

The headline must be the strongest element on the left side. The image collage should support the emotional storytelling without overpowering the text.

## 10. Spacing and Alignment Rules

Use consistent spacing based on an 8px grid system.

Recommended spacing:

* Header top padding: 32–40px
* Gap between eyebrow and headline: 24–32px
* Gap between headline and paragraph: 24–32px
* Gap between paragraph and CTA buttons: 36–48px
* Gap between CTA buttons: 20–28px
* Image card overlap: controlled and intentional, not random

Keep alignment clean:

* Left text should align consistently
* Buttons should align with the text block
* Image collage should align visually with the hero center
* Avoid cramped spacing

## 11. Responsive Layout Rules

For tablet:

* Keep the two-column layout if space allows
* Reduce headline size to around 56–72px
* Reduce image collage complexity slightly

For mobile:

* Stack the layout vertically
* Header becomes simplified
* Text appears first, image collage below
* Headline size around 42–52px
* CTA buttons can stack vertically or stay side-by-side if enough space
* Reduce decorative elements
* Keep image collage simpler with fewer overlapping cards

## 12. Design Quality Rules

The final design should look like a real tourism website created by an experienced UI/UX designer.

Do:

* Use realistic imagery
* Keep the layout spacious
* Maintain visual balance
* Use refined typography
* Use soft shadows and rounded corners
* Make the design feel warm, premium, and authentic

Do not:

* Make the design look like a generic AI template
* Use random floating elements
* Use excessive gradients
* Add too many icons
* Use unrealistic glossy effects
* Overcrowd the image collage
* Change the visual focus away from the village tourism experience
